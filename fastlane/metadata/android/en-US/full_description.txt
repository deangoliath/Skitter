A FOSS application used for navigating the public transit systems using live GTFS data.

- Favorite Routes
- Saved Locations
- Manage Tickets & Passes
- Advanced Trip Planning

Currently, not all cities are covered. This app will help with the Lynx bus system in Orlando, Florida.

Want this for your city? It really is very easy! Open an feature request issue on the source code repository.

This is not a map app. Please use OSMAND+ available on f-droid.org.
